# Product Goals Spec

## Purpose

Position Synthetix as a preliminary survey simulation sandbox, not a substitute for representative human surveys.

## Required Behavior

- Present Synthetix as a tool for exploring likely response patterns before real fieldwork.
- Make students, researchers, founders, marketers, and professionals first-class audiences.
- Keep segmentation, respondent logic, synthetic responses, and aggregation inspectable.
- State clearly that synthetic results are directional simulation evidence.

## Acceptance Criteria

- `goals.md` exists and includes positioning, primary objective, core goals, ground-truth policy, report quality goal, non-goals, and current readiness.
- Product-facing copy never claims replacement of human surveys.
- Reports and UI must use "simulation" language rather than "polling result" language.

## Agent Allocation

- Assigned model: `gpt-5.4`
- Reason: product positioning and scientific boundary decisions require high judgment.
